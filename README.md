# Wenyu Zhu - Academic Personal Website

This is a modern, responsive academic personal website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Professional scroll animations and transitions
- **Modern UI**: Clean, academic-style interface with professional color scheme
- **Easy Navigation**: Fixed navigation bar with smooth scrolling
- **Comprehensive Sections**:
  - About/Hero section with profile photo
  - Research interests
  - Publications (journals, conferences, workshops)
  - Professional experience timeline
  - Education and awards
  - Technical skills
  - Contact information

## Setup Instructions

1. **Add Your Photo**: 
   - Place your photo file named `zhuwenyu.jpeg` in the same directory as `index.html`
   - If you don't have the photo yet, a placeholder will be shown

2. **Open the Website**:
   - Simply double-click `index.html` to open it in your default browser
   - Or right-click and select "Open with" to choose a specific browser

3. **Deploy Online** (Optional):
   You can host this website for free using:
   - **GitHub Pages**: Push to a GitHub repository and enable GitHub Pages
   - **Netlify**: Drag and drop the folder to Netlify
   - **Vercel**: Deploy directly from GitHub
   - **Google Firebase Hosting**: Use Firebase CLI

## Customization

### Colors
Edit the CSS variables in `styles.css` to change the color scheme:
```css
:root {
    --primary-color: #AA5733;  /* Main accent color */
    --secondary-color: #2c3e50; /* Headers */
    --accent-color: #e74c3c;   /* Links hover */
}
```

### Content
All content is in `index.html`. You can edit:
- Personal information
- Research interests
- Publications
- Experience details
- Education information
- Skills and tools

### Adding New Sections
Follow the existing HTML structure to add new sections with the `.section` class.
